# Fraud Detection & Transaction Monitoring Platform with MCP Integration

## Project Overview

This project is a production-grade portfolio application designed to demonstrate:

- microservices architecture
- asynchronous processing
- fraud detection rules
- PostgreSQL persistence
- Redis-based fast state usage
- RabbitMQ event-driven messaging
- real-time dashboard updates
- MCP integration for AI agent tools

## Core Services

### Backend Services
- transaction-api-service
- fraud-worker-service
- mcp-server-service

### Frontend
- frontend-web

## Infrastructure

- docker-compose
